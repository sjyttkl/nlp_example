# bi-lstm-crf-ner-tf2.0

Named Entity Recognition (NER) task using Bi-LSTM-CRF model implemented in Tensorflow2.0.


# Requirements

- python >3.6
- tensorflow==2.0.0
- tensorflow-addons==0.6.0
